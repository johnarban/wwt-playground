WWT examples

The base for these examples comes from the CosmicDS Vue Template, heavily modified to use Vite, as well as updated version of Vuetify and Vue.

  1. Empty Project
    Project Page GitHub Repo
    The blank WWT project demonstrates using WWT with Vite and merges (in an opinionated way) the linting that is default on the vue-ds-template and the defaults from a project created with npx init vuetify.
  2. SPHERE-X All Sky True Crossfade
    Project Page GitHub Repo
    Demonstrates a true cross-fade using the SPHEREx all sky Star and Line emission images. by true cross-fade where the total opacity at the midpoint is 100%, as opposed to a linear cross-fade where in the middle the total opacity is 75% (100% - 50%×50%).
  3. 3D FITS cube w/ velocity slider
    Project Page GitHub Repo
    Demonstrate loading a 3D fits cube into WWT. This project uses a hack to trick the WWT engine into dispalying the appropriate data plane by manually paging through the FITS data. It only works for PPV oriented cubes, with a TAN projection, and really seems to need a properly formatted FITS header
Having multiple WWT panes open may cause the browser to slow down
